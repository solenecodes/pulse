# Pulse Merchandising

Keep the storefront premium and editorial. Lead with the strongest lifestyle asset, then use product categories to reduce choice friction.

Checklist:
- Preserve the Recovery hero unless the user asks for a different campaign.
- Keep categories short: Core Line, Zero Sugar, Recovery, Limited Drops.
- Put the strongest everyday products first inside each category.
- Avoid supplement-store language, clinical claims, and busy promotional blocks.
